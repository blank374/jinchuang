## 1. 必读论文

### 1. DINOv3

- 论文：DINOv3
- 时间：2025
- 链接：https://arxiv.org/abs/2508.10104
- 本地文件：`papers/DINOv3_2025.pdf`
- 对赛题的用处：作为强视觉特征提取器，适合图像相似度、细粒度视觉差异、近重复检测。可与 CLIP/SigLIP2 做特征融合。

### 2. SigLIP 2

- 论文：SigLIP 2: Multilingual Vision-Language Encoders with Improved Semantic Understanding, Localization, and Dense Features
- 时间：2025
- 链接：https://arxiv.org/abs/2502.14786
- 本地文件：`papers/SigLIP2_2025.pdf`
- 对赛题的用处：多语言图文编码器，适合中文类别提示词、影像分类、图文检索和通用 embedding 提取。

### 3. ELIP

- 论文：ELIP: Enhanced Visual-Language Foundation Models for Image Retrieval
- 时间：2025
- 链接：https://arxiv.org/abs/2502.15682
- 本地文件：`papers/ELIP_Image_Retrieval_2025.pdf`
- 对赛题的用处：专门面向图像检索增强视觉语言模型，可作为报告里“检索增强方向”的近期研究依据。

### 4. Transductive Learning for Near-Duplicate Image Detection

- 论文：Transductive Learning for Near-Duplicate Image Detection in Scanned Photo Collections
- 时间：2024
- 链接：https://arxiv.org/abs/2410.19437
- 本地文件：`papers/Near_Duplicate_Image_Detection_Transductive_2024.pdf`
- 对赛题的用处：和本题的“历史影像近重复检测”非常贴近，尤其适合讨论目标数据集可提前获取时，如何用自监督/转导学习适配目标域。

### 5. CLIP

- 论文：Learning Transferable Visual Models From Natural Language Supervision
- 时间：2021
- 链接：https://arxiv.org/abs/2103.00020
- 本地文件：`papers/CLIP_2021.pdf`
- 对赛题的用处：赛题说明中明确提到 CLIP，是最基础的多模态预训练模型基线。

### 6. FAISS

- 论文：Billion-scale similarity search with GPUs
- 时间：2017
- 链接：https://arxiv.org/abs/1702.08734
- 本地文件：`papers/FAISS_Billion_Scale_Similarity_Search_2017.pdf`
- 对赛题的用处：用于大规模向量检索。系统需要与历史影像比对时，FAISS 是最合适的工程组件之一。

## 3. 方法选型建议

### 分类模型

首选：

```text
SigLIP2 / Chinese-CLIP / CLIP image encoder
  -> frozen embedding
  -> Logistic Regression / MLP
```

如果面签照片类别和其他类别差异明显，冻结 embedding 就可能有不错效果。

### 相似度模型

首选：

```text
DINOv3 或 SigLIP2 embedding
  -> L2 normalize
  -> cosine similarity
  -> FAISS TopK
```

增强：

```text
embedding 融合 + 感知哈希 + 度量学习微调
```

### 阈值策略

必须基于验证集做：

```text
PR 曲线
ROC 曲线
F1 最优阈值
业务复核量约束下的阈值
```

报告里要强调：风控场景倾向高召回，再通过人工复核消化误报。

## 4. 可写进报告的创新点

- 用多模态模型实现“文本提示 + 图像分类”的面签照片筛选。
- 用 DINOv3/SigLIP2 等最新视觉基础模型替代传统像素比对。
- 引入 FAISS 向量库，实现历史影像快速 TopK 检索。
- 结合度量学习，让相似面签照片在 embedding 空间内更聚集。
- 用阈值分层策略输出高/中/低风险可疑交易。
- 通过可视化页面展示可疑影像对、相似度分数和业务关联信息。

## 5. 第一版实验清单

建议按以下顺序做实验：

1. CLIP embedding + 线性分类器，得到影像分类 baseline。
2. SigLIP2 embedding + 线性分类器，对比分类指标。
3. DINOv3 embedding + cosine similarity，对比相似图片 TopK 命中。
4. CLIP/SigLIP2/DINOv3 三种 embedding 的相似度 AUC、Recall@K。
5. 加入 FAISS，测试批量检索耗时。
6. 做不同阈值下的 Precision、Recall、F1。
7. 若时间允许，做 Triplet Loss 或 Supervised Contrastive 微调。
